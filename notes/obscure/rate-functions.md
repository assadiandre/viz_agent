# Obscure Rate Functions
> Manim v0.20.1

These are exported but rarely used. `smooth` covers most use cases.

---

| Function | Description |
|---|---|
| `smoothstep` | 1st order smoothstep — zero velocity at both ends |
| `smootherstep` | 2nd order smoothstep — zero velocity and acceleration at both ends |
| `smoothererstep` | 3rd order smoothstep — zero velocity, acceleration, and jerk at both ends |
| `running_start` | Starts with a small overshoot backward before moving forward |
